# Mean

The mean is the average value of a dataset and is one of the most commonly used measures of central tendency.

---

## Formula

### Population Mean

```text
      Σx
μ = ----
      N
```

Where:

* `Σx` = Sum of all observations
* `N` = Total number of observations
* `μ` = Population Mean

### Sample Mean

```text
      Σx
x̄ = ----
       n
```

Where:

* `x̄` = Sample Mean
* `n` = Sample Size

---

# Example 1: Mean of Ungrouped Data

## Data

| x | 30 | 32 | 32 | 37 | 42 | 50 | 50 |
|---|----|----|----|----|----|----|----|

## Calculation

```text
x̄ = (30 + 32 + 32 + 37 + 42 + 50 + 50) / 7

x̄ = 273 / 7

x̄ = 39
```

## Answer

**Mean = 39**

# Example 2: Mean Using Frequency Distribution

## Data

| Age (x) | Frequency (f) | f × x |
| ------- | ------------- | ----- |
| 14      | 76            | 1064  |
| 15      | 121           | 1815  |
| 16      | 118           | 1888  |
| 17      | 101           | 1717  |
| 18      | 54            | 972   |

## Formula

```text
        Σfx
x̄ = ----------
        Σf
```

## Calculation

```text
Σfx = 1064 + 1815 + 1888 + 1717 + 972

Σfx = 7456

Σf = 76 + 121 + 118 + 101 + 54

Σf = 470

x̄ = 7456 / 470

x̄ = 15.9
```

## Answer

**Mean Age ≈ 15.9 years**

# Example 3: Mean of Grouped Data

## Weight Distribution

| Class Interval | Frequency (f) | Midpoint (x) |
| -------------- | ------------- | ------------ |
| 4–6            | 3             | 5            |
| 7–8            | 11            | 7.5          |
| 9–10           | 14            | 9.5          |
| 10–15          | 6             | 12.5         |

## Formula

```text
        Σfx
x̄ = ----------
        Σf
```

## Calculation

```text
Σfx = 3(5) + 11(7.5) + 14(9.5) + 6(12.5)

    = 15 + 82.5 + 133 + 75

    = 305.5

Σf = 3 + 11 + 14 + 6

   = 34

x̄ = 305.5 / 34

x̄ = 8.98
```

## Answer

**Mean Weight ≈ 8.98 kg**

# Example 4: Mean Rainfall

## Data

| Rainfall (cm) | Frequency (f) | Midpoint (x) |
| ------------- | ------------- | ------------ |
| 0–2           | 96            | 1            |
| 2–5           | 18            | 3.5          |
| 5–10          | 7             | 7.5          |
| 10–30         | 4             | 20           |
| 30+           | 0             | -            |

## Calculation

```text
Σfx = 96(1) + 18(3.5) + 7(7.5) + 4(20)

    = 96 + 63 + 52.5 + 80

    = 291.5

Σf = 96 + 18 + 7 + 4

   = 125

x̄ = 291.5 / 125

x̄ = 2.33
```

## Answer

**Average Rainfall ≈ 2.33 cm**

---

## Issues in Example 4

### 1. Open-Ended Class Interval

The interval `30+` has no upper limit, so its midpoint cannot be determined. If its frequency were non-zero, the grouped mean could not be calculated directly.

**Fix:** Replace `30+` with a closed interval such as `30–50`.

### 2. Wide Class Interval

The interval `10–30` is much wider than the other intervals. Using a single midpoint (`20`) may not accurately represent all observations in this range.

**Fix:** Divide the interval into smaller classes such as `10–15`, `15–20`, `20–25`, and `25–30` to improve accuracy.

# Quick Summary

## Mean

```text
            Sum of Observations
Mean = -----------------------------
       Number of Observations
```

### Ungrouped Data

```text
      Σx
x̄ = ----
       n
```

### Frequency Distribution

```text
        Σfx
x̄ = ----------
        Σf
```

### Steps for Grouped Data

1. Find class midpoints.
2. Calculate `f × x`.
3. Compute `Σfx`.
4. Compute `Σf`.
5. Apply:

```text
        Σfx
x̄ = ----------
        Σf
```

---

# Key Takeaways

* Mean is the arithmetic average of a dataset.
* For grouped data, class midpoints are used as representative values.
* Open-ended intervals may create problems because a midpoint cannot be determined.
* A very high frequency in one class can significantly affect the estimated mean.
* Unequal class widths are allowed but may reduce accuracy.
* Grouped-data means are approximations and may differ from the true mean calculated from raw data.
* Proper class boundaries prevent overlap between intervals.
