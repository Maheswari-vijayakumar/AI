

### Example 1: Ungrouped Data

```python
x = [30, 32, 32, 37, 42, 50, 50]

mean = sum(x) / len(x)

print("Mean =", mean)
```

Output:

```text
Mean = 39.0
```

### Example 2: Frequency Distribution

```python
age = [14, 15, 16, 17, 18]
freq = [76, 121, 118, 101, 54]

sum_fx = sum(x * f for x, f in zip(age, freq))
sum_f = sum(freq)

mean = sum_fx / sum_f

print("Σfx =", sum_fx)
print("Σf =", sum_f)
print("Mean =", round(mean, 2))
```

Output:

```text
Σfx = 7456
Σf = 470
Mean = 15.86
```

### Example 3: Grouped Data

```python
midpoint = [5, 7.5, 9.5, 12.5]
freq = [3, 11, 14, 6]

sum_fx = sum(x * f for x, f in zip(midpoint, freq))
sum_f = sum(freq)

mean = sum_fx / sum_f

print("Mean =", round(mean, 2))
```

Output:

```text
Mean = 8.99
```

### Generic Function (Best for Notes)

```python
def mean_grouped(midpoints, frequencies):
    sum_fx = sum(x * f for x, f in zip(midpoints, frequencies))
    sum_f = sum(frequencies)
    return sum_fx / sum_f

mean = mean_grouped(
    [1, 3.5, 7.5, 20],
    [96, 18, 7, 4]
)

print(round(mean, 2))
```

This mirrors the statistics formula directly:

```text
Σfx
----
 Σf
```

where:

```python
sum_fx = sum(x * f for x, f in zip(midpoints, frequencies))
sum_f = sum(frequencies)

mean = sum_fx / sum_f
```
