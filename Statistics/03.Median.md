
## 1. Median is the Middle Value of an Ordered Dataset

The median divides an ordered dataset into two equal halves. Half of the observations lie below the median and half lie above it.

### Example

```text
30, 32, 32, 37, 42, 50, 50
```

The middle value is:

```text
37
```

Therefore:

```text
Median = 37
```

### Note

The data must be sorted before finding the median.

---

## 2. For Odd n, Select the Middle Observation

When the number of observations is odd, there is exactly one middle value.

### Formula

```text
Median Position = (n + 1) / 2
```

### Example

```text
30, 32, 32, 37, 42, 50, 50
```

```text
n = 7

Median Position

= (7 + 1)/2

= 4
```

The 4th observation is:

```text
37
```

Therefore:

```text
Median = 37
```

### Note

No averaging is required because there is a single middle observation.

---

## 3. For Even n, Average the Two Middle Observations

When the number of observations is even, there is no single middle value.

### Example

```text
30, 32, 32, 37, 42, 50
```

```text
n = 6
```

Middle observations:

```text
3rd = 32
4th = 37
```

Median:

```text
(32 + 37)/2

= 34.5
```

### Note

The median is always the average of the two central observations.

---

## 4. Frequency Tables Use Cumulative Frequencies to Locate the Median Position

For frequency distributions, individual observations are not listed. Therefore, cumulative frequencies are used to identify where the median lies.

### Example

| Age | Frequency | Cumulative Frequency |
|------|----------:|--------------------:|
| 14 | 76 | 76 |
| 15 | 121 | 197 |
| 16 | 118 | 315 |
| 17 | 107 | 422 |
| 18 | 54 | 476 |

```text
Total Frequency = 476

Median Position

= (476 + 1)/2

= 238.5
```

Since:

```text
197 < 238.5 ≤ 315
```

the median lies in the age-16 group.

### Note

Cumulative frequencies help identify the median class quickly.

---

## 5. Grouped-Data Medians Are Estimated Using Interpolation

For grouped data, the exact median value is unknown because individual observations are not available.

Instead, interpolation is used to estimate the median.

### Formula

```text
Median = L + [(N/2 - CF)/f] × h
```

Where:

```text
L  = Lower boundary of median class
CF = Cumulative frequency before median class
f  = Frequency of median class
h  = Class width
```

### Example

Rainfall Data:

| Rainfall (mm) | Frequency |
|--------------|----------:|
| 0–2 | 96 |
| 2–5 | 18 |
| 5–10 | 7 |
| 10–30 | 4 |

Estimated Median:

```text
≈ 1.3 mm
```

### Note

Interpolation provides an estimate rather than an exact value.

---

## 6. Interpolation Assumes a Uniform Distribution Within the Median Class

Interpolation assumes that observations are spread evenly throughout the median class interval.

### Example

Median class:

```text
0–2 mm
```

Assumption:

```text
0.1, 0.5, 1.0, 1.5, 1.9
```

are evenly distributed throughout the interval.

### Note

If observations are clustered near one end of the interval, the estimate may not accurately represent the true median.

---

## 7. Wider Class Intervals Produce Less Accurate Median Estimates

Large intervals represent a wide range of possible values using a single estimate.

### Example

```text
10–30
```

Width:

```text
20
```

This interval could contain:

```text
10, 11, 12
```

or

```text
25, 27, 29
```

Both situations produce different true medians.

### Note

Smaller class intervals generally produce more accurate median estimates because they capture the data distribution more precisely.
