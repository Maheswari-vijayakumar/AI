
# Mode

The mode is the value that occurs most frequently in a dataset.

Unlike the mean and median, the mode identifies the most common observation rather than the center of the data.

---

## Notation

```text
Mode
Mo
```

---

# Example 1: Single Mode (Unimodal)

## Data

| x | 30 | 32 | 32 | 37 | 42 | 50 |
|---|---|---|---|---|---|---|

### Frequency of Each Value

| Value | Frequency |
|---------|---------:|
| 30 | 1 |
| 32 | 2 |
| 37 | 1 |
| 42 | 1 |
| 50 | 1 |

The value:

```text
32
```

appears most frequently.

### Answer

```text
Mode = 32
```

---

## Python Example

```python
from collections import Counter

data = [30, 32, 32, 37, 42, 50]

freq = Counter(data)

mode = max(freq, key=freq.get)

print("Mode =", mode)
```

Output:

```text
Mode = 32
```

---

# Example 2: Two Modes (Bimodal)

## Data

| x | 30 | 32 | 32 | 37 | 42 | 50 | 50 |
|---|---|---|---|---|---|---|---|

### Frequency of Each Value

| Value | Frequency |
|---------|---------:|
| 30 | 1 |
| 32 | 2 |
| 37 | 1 |
| 42 | 1 |
| 50 | 2 |

The values:

```text
32
50
```

occur with the same highest frequency.

### Answer

```text
Modes = 32, 50
```

This dataset is called:

```text
Bimodal
```

because it has two modes.

---

# Example 3: No Mode

## Data

| x | 30 | 32 | 37 | 42 | 50 |
|---|---|---|---|---|---|

Every value occurs exactly once.

### Frequency Table

| Value | Frequency |
|---------|---------:|
| 30 | 1 |
| 32 | 1 |
| 37 | 1 |
| 42 | 1 |
| 50 | 1 |

No value occurs more frequently than the others.

### Answer

```text
No Mode
```

---

# Types of Modes

## Unimodal

One mode.

```text
30, 32, 32, 37, 42

Mode = 32
```

---

## Bimodal

Two modes.

```text
30, 32, 32, 37, 50, 50

Modes = 32, 50
```

---

## Multimodal

More than two modes.

```text
10, 10, 20, 20, 30, 30
```

Modes:

```text
10, 20, 30
```

---

## No Mode

No value repeats.

```text
10, 20, 30, 40, 50
```

---

# Mode for Qualitative (Categorical) Data

One advantage of mode is that it can be used with non-numeric data.

## Example

| Colour | Frequency |
|---------|---------:|
| Red | 43 |
| Yellow | 22 |
| Green | 35 |
| Blue | 51 |

The highest frequency is:

```text
51
```

corresponding to:

```text
Blue
```

### Answer

```text
Mode = Blue
```

---

## Why Mean and Median Cannot Be Used Here

Colours cannot be added, divided, or ordered meaningfully.

```text
Red + Blue = ?
```

Therefore:

```text
Mean   ❌
Median ❌
Mode   ✅
```

The mode is often the only measure of central tendency applicable to purely categorical data.

---

# Mode from a Frequency Table

## Example

| Number of Cars | Frequency (Millions) |
|---------------|---------------------:|
| 0 | 4.3 |
| 1 | 5.9 |
| 2 | 2.6 |
| 3 | 1.3 |
| 4+ | 0.6 |

Highest frequency:

```text
5.9 million
```

corresponds to:

```text
1 car
```

### Answer

```text
Mode = 1
```

### Interpretation

The most common number of cars owned by a household is:

```text
1 car
```

# Mode in a Distribution Curve

The mode corresponds to the highest point (peak) of a distribution.

```text
Highest Frequency
       ▲
       │
       ●  ← Mode
      / \
     /   \
----/-----\-----------
```

### Interpretation

The mode represents the most frequently occurring value or category.

---

# Relationship Between Mean, Median, and Mode

The relative positions of the mean, median, and mode depend on the shape of the distribution.

---

## Right-Skewed Distribution (Positive Skew)

```text
Mode < Median < Mean
```

Example:

```text
Income
House Prices
Wealth
```

A few very large values pull the mean to the right.

```text
Mode ----- Median -------- Mean
```

---

## Left-Skewed Distribution (Negative Skew)

```text
Mean < Median < Mode
```

A few very small values pull the mean to the left.

```text
Mean ----- Median -------- Mode
```

---

## Symmetric Distribution

```text
Mean = Median = Mode
```

Example:

```text
Normal Distribution
```

All three measures occur at the center.

---

# Why Does the Mean Move More Than the Median?

The mean uses every observation:

```text
Mean = Σx / n
```

Therefore, extreme values strongly affect it.

The median depends only on position:

```text
Middle Observation
```

and is less sensitive to outliers.

The mode depends only on frequency:

```text
Most Common Value
```

and may remain unchanged even when extreme values are added.

---

# Python Examples

## Single Mode

```python
from statistics import mode

data = [30, 32, 32, 37, 42, 50]

print(mode(data))
```

Output:

```text
32
```

---

## Multiple Modes

```python
from statistics import multimode

data = [30, 32, 32, 37, 42, 50, 50]

print(multimode(data))
```

Output:

```text
[32, 50]
```

---

# Key Takeaways

## 1. Mode is the Most Frequently Occurring Value

The mode identifies the observation that appears most often in a dataset.

Example:

```text
30, 32, 32, 37, 42
```

Mode:

```text
32
```

---

## 2. A Dataset Can Have More Than One Mode

If multiple values share the highest frequency, the dataset can be bimodal or multimodal.

Example:

```text
30, 32, 32, 50, 50
```

Modes:

```text
32, 50
```

---

## 3. Some Datasets Have No Mode

If every observation occurs the same number of times, there is no mode.

Example:

```text
10, 20, 30, 40
```

---

## 4. Mode Works with Qualitative Data

Mode can be calculated for categories such as:

```text
Colour
Gender
Brand
Country
```

where mean and median may not be meaningful.

---

## 5. The Mode Represents the Peak of a Distribution

In a graph, the mode occurs at the point of highest frequency.

The peak of the distribution corresponds to the most common value.

---

## 6. Mean, Median, and Mode Change with Distribution Shape

```text
Right Skewed:
Mode < Median < Mean

Left Skewed:
Mean < Median < Mode

Symmetric:
Mean = Median = Mode
```

Understanding this relationship helps describe the shape of a dataset.
